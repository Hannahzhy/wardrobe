📦 数据管理
☁️ 云端同步 未连接
连接 Supabase 数据库后，所有设备自动同步数据。手机上传的照片会立刻出现在电脑上。
💡 如何获取？去 supabase.com 创建免费项目 → Settings → API → 复制 URL 和 anon key。SQL 建表语句见下方。
📤 导出数据
将所有衣服照片和搭配方案打包下载为一个文件，可用来备份或在其他设备上导入。
📥 导入数据
从之前导出的文件恢复数据。注意：导入会合并到当前数据，不会覆盖已有内容。
🗑 清除数据
删除所有衣服和搭配方案，此操作不可恢复。建议先导出一份备份。
